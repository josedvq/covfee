---
title: Getting Started
---

This guide takes you through the process of creating a simple Covfee study with four tasks:

1. a demographics questionnaire
2. an instructions screen loaded from a Markdown file
3. a continuous annotation task on a video
4. a final feedback questionnaire

## 1. Install Covfee

Follow the [installation instructions](installation) if you have not done so already.

## 2. Create the project folder

First create a folder for your Covfee project. This folder may be anywhere in the file system. Inside it, create the following structure:

```text
myproject
    getting_started.py
    local.env
    www
        instructions.md
```

Covfee expects local project assets to live in a folder named `www` directly under the project folder.

If you want to override the default host, port, admin credentials, or other settings, create a `local.env` file in the project root. Covfee loads `local.env` automatically for the default local mode.

## 3. Add the study file

Paste the following into `getting_started.py`:

```python
from covfee import HIT, CovfeeApp, Project, tasks


task_demographics = tasks.QuestionnaireTaskSpec(
    name="Demographics",
    form={
        "fields": [
            {
                "name": "age",
                "label": "Your age:",
                "input": {"inputType": "InputNumber"},
            },
            {
                "name": "sex",
                "label": "Your sex:",
                "input": {
                    "inputType": "Select",
                    "options": [
                        {"label": "Male", "value": "m"},
                        {"label": "Female", "value": "f"},
                    ],
                },
            },
            {
                "name": "nationality",
                "label": "Your nationality:",
                "input": {"inputType": "Input"},
            },
        ]
    },
)

task_instructions = tasks.InstructionsTaskSpec(
    name="Instructions",
    content={"type": "link", "url": "$$www$$/instructions.md"},
)

task_annotation = tasks.ContinuousAnnotationTaskSpec(
    name="Arousal annotation",
    annotations=[{"name": "Arousal", "interface": "RankTrace"}],
    media={
        "type": "video",
        "url": "https://cdn.jsdelivr.net/gh/josedvq/covfee@master/samples/keypoints/www/person_reading.mp4",
    },
    userCanAdd=False,
)

task_feedback = tasks.QuestionnaireTaskSpec(
    name="Feedback",
    form={
        "fields": [
            {
                "name": "rating",
                "label": "How would you rate your experience in completing this experiment?",
                "required": True,
                "input": {"inputType": "Rate", "allowHalf": True},
            },
            {
                "name": "feedback",
                "label": "Do you have any comments that can help us improve the experience?",
                "input": {"inputType": "Input.TextArea"},
            },
        ]
    },
)

hit = HIT("Basic annotation")
hit.add_journey(
    nodes=[
        task_demographics,
        task_instructions,
        task_annotation,
        task_feedback,
    ]
)

projects = [Project("My Covfee Project", email="example@example.com", hits=[hit])]
app = CovfeeApp(projects)
```

Covfee looks for a module-level object named `app`, so that name is important.

## 4. Add the instructions file

Paste the following into `www/instructions.md`:

```md
# Instructions

You will be shown a short video. Please rate the speaker's arousal using your keyboard up and down arrows.
```

The `$$www$$` variable in the Python file points to this special `www` folder.

Optional `local.env`:

```bash
COVFEE_HOST=localhost
COVFEE_PORT=5001
```

:::tip
This example uses a publicly hosted sample video so the project stays lightweight. If you prefer a local file, place it in `www/` and change the media URL to something like `$$www$$/person_reading.mp4`.
:::

## 5. Run Covfee

From the project folder, run:

```bash
cd myproject
covfee make getting_started.py
```

This will parse the study, initialize the database, and start Covfee. The admin panel should be available at [http://localhost:5001/admin#](http://localhost:5001/admin#) by default.

If you edit the study and want to rebuild the database, rerun:

```bash
covfee make getting_started.py --force
```

## 6. Annotate

You can now use the admin panel to open HIT URLs, monitor progress, and download completed annotations.
